# GitHub Repository Finder

A simple web app that helps you discover random GitHub repositories based on a selected programming language. Choose a language, fetch a random repo, and instantly see key details like its name, description, star count, and fork count. Want another suggestion? Just hit Refresh to load a new random repository using the same language.

---

## Features

- Select a programming language from a dropdown menu
- Fetch a random GitHub repository written in that language
- Displays essential repo details:
  - Repository name
  - Description
  - Star count
  - Fork count
- Refresh button to get a new repo without reselecting the language
- Fast and responsive UI
- Error Handling
